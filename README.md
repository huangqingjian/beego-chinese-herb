# beego-chinese-herb

作为一个golang初学者，基于beego框架搭建的一个中药类后端管理系统，内置了一个服务模块全部基本功能，包括但不限于数据库、redis、i18n、异步、全局异常处理等。
ps.学习过程中发现国内golang方面的文档确实比较少，特别是那种系统性的文档

### 服务说明

参看conf/app.conf配置文件，修改对应配置后，导入db.sql即可
